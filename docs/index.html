What is pypacket ?

A python module that can make files packet for send a request file to a server or create a log.

Create a .json packet

To create a .json packet we are using the class json and the function 'create_json_packet()' then the code look's like this :

        1 from pypacket import *
        2    
        3 json.create_json_packet("Hello.json", json_content='"msg" : "Hello World !"')

Create a .txt packet

To create a .txt packet we are using the class txt and the function 'create_txt_packet()' then the code look's like this :

        1 from pypacket import *
        2     
        3 txt.create_txt_packet("Hello.txt", txt_content='Hello World !')

How to install it ?
