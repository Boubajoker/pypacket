What is pypacket ?

A python module that can make files packet for send a request file to a server or create a log.

Create a .json packet

To create a .json packet we are using the class json and the function 'create_json_packet()' then the code look's like this :

        1 from pypacket import *
        2    
        3 Json.create_json_packet("Hello.json", json_content='"msg" : "Hello World !"')

Create a .txt packet

To create a .txt packet we are using the class txt and the function 'create_txt_packet()' then the code look's like this :

        1 from pypacket import *
        2     
        3 Txt.create_txt_packet("Hello.txt", txt_content='Hello World !')

Create a .xml packet

To create a .xml packet we are using the class xml and the function 'create_xml_packet()' then the code look's like this :

        1 from pypacket import *
        2     
        3 Xml.create_txt_packet("Hello.xml", xml_content='< template >Hello World !< / template >')

Create a .js packet

To create a .js packet we are using the class JavaScript and the function 'create_js_packet()' then the code look's like this :

        1 from pypacket import *
        2     
        3 JavaScript.create_txt_packet("Hello.js", js_content='Hello World !')

read a packet file

to read a packet file just use the class Read and the function 'file_content()' then the code look's like this:

        1 from pypacket import *
        2     
        3 Read.file_content("Hello.txt")
